# Politique de Confidentialité - Kalimat

**Dernière mise à jour : 6 février 2026**

## Introduction

Bienvenue dans Kalimat ("l'Application"). Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations lorsque vous utilisez notre application de lecture du Coran.

## Données Collectées

### Données stockées localement
L'Application stocke les données suivantes **uniquement sur votre appareil** :
- Vos préférences d'affichage (thème, langue)
- Votre progression de lecture
- Vos versets favoris
- Vos statistiques de lecture

### Données anonymes synchronisées
Pour améliorer l'Application, nous collectons de manière **anonyme** :
- Un identifiant d'appareil généré aléatoirement (non lié à votre identité)
- Des statistiques de lecture agrégées (nombre de versets lus)

Ces données ne permettent **pas** de vous identifier personnellement.

## Données NON Collectées

Nous ne collectons **jamais** :
- Votre nom, email ou informations personnelles
- Votre localisation
- Vos contacts
- Aucune donnée de paiement

## Utilisation des Données

Les données anonymes sont utilisées uniquement pour :
- Améliorer les fonctionnalités de l'Application
- Comprendre comment l'Application est utilisée

## Partage des Données

Nous ne vendons, n'échangeons ni ne transférons vos données à des tiers.

## Sécurité

Nous utilisons le chiffrement HTTPS standard pour toute transmission de données.

## Droits des Utilisateurs

Vous pouvez à tout moment :
- Réinitialiser vos données locales via les Paramètres de l'Application
- Désinstaller l'Application pour supprimer toutes les données

## Services Tiers

L'Application utilise :
- **Expo** pour le framework de développement
- **Cloudflare** pour le stockage des statistiques anonymes
- **API Quran.com** pour les récitations audio

## Modifications

Nous pouvons mettre à jour cette politique. Les modifications seront publiées dans l'Application.
